# Специфікація екранів: Функціонал телефонних дзвінків

**Дата:** 2025-03-13  
**Контекст:** Прототип екранів дзвінків (Contact list → Profile → Call).  
**User job:** Швидко знайти контакт і подзвонити йому.

---

## User flow

**Список контактів** → (скрол / пошук) → **вибір контакту** → **Профіль контакту** → **Подзвонити** → **Екран дзвінка** → **Завершити дзвінок** → повернення до **списку контактів**.

---

## 1. Список контактів (Contact list)

### Призначення
Перегляд і пошук контактів перед вибором контакту для дзвінка.

### Елементи екрану
| Елемент | Опис |
|--------|------|
| Заголовок | Текст «Контакти» |
| Поле пошуку | Placeholder: «Пошук за ім'ям або номером»; введення фільтрує список |
| Список контактів | Скролований список; кожен рядок: аватар/ініціали, ім'я, опційно номер; клік по рядку відкриває профіль |

### Поведінка
- **Пошук:** фільтрація списку по введеному тексту (ім'я або номер) у реальному часі.
- **Скрол:** вертикальний скрол списку контактів.
- **Вибір контакту:** клік по рядку контакту → навігація на екран «Профіль контакту».

---

## 2. Профіль контакту (Contact profile)

### Призначення
Перегляд деталей обраного контакту та ініціація дзвінка.

### Елементи екрану
| Елемент | Опис |
|--------|------|
| Кнопка «Назад» | Стрілка або текст «Назад»; повертає до списку контактів |
| Аватар/ініціали | Візуал контакту (коло з ініціалами, якщо немає фото) |
| Ім'я контакту | Відображається ім'я обраного контакту |
| Номер телефону | Основний номер контакту |
| Кнопка «Подзвонити» | Call-to-action; ініціює дзвінок → перехід на екран дзвінка |

### Поведінка
- **Назад:** повернення до екрану «Список контактів» (стан пошуку зберігається, якщо реалізовано).
- **Подзвонити:** перехід на екран «Екран дзвінка» з даними обраного контакту.

---

## 3. Екран дзвінка (Call screen)

### Призначення
Відображення активного дзвінка та можливість завершити дзвінок.

### Елементи екрану
| Елемент | Опис |
|--------|------|
| Ім'я контакту | Ім'я абонента або «Невідомий», якщо немає в контактах |
| Номер телефону | Номер, на який виконується дзвінок |
| Статус/тривалість | Текст «Дзвінок…» або таймер у форматі «00:00» |
| Кнопка «Завершити дзвінок» | Червона кнопка внизу екрану; завершує дзвінок |

### Поведінка
- **Завершити дзвінок:** клік по кнопці → повернення на екран «Список контактів» (початкова точка флоу).

---

## Критерії готовності (прототип)

- [ ] Відкривається список контактів з пошуком і скролом.
- [ ] Пошук фільтрує контакти по імені та/або номеру.
- [ ] Клік по контакту відкриває профіль контакту.
- [ ] У профілі є кнопка «Подзвонити», клік відкриває екран дзвінка.
- [ ] На екрані дзвінка відображаються ім'я та номер; кнопка «Завершити» повертає до списку.
- [ ] Кнопка «Назад» у профілі повертає до списку контактів.
