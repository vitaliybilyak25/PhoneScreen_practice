# PhoneScreen_practice

Phone screen prototyping practice project: інтерактивний прототип екранів телефонних дзвінків (список контактів → профіль контакту → екран дзвінка).

## Як відкрити прототип

1. Відкрийте файл `index.html` у браузері (подвійний клік або перетягнути в вікно браузера).
2. Або запустіть локальний сервер у корені проєкту, наприклад:
   - `npx serve .` або `python3 -m http.server 8000` — потім відкрийте `http://localhost:8000` (або вказаний порт).

## User flow

- **Список контактів** — пошук за ім'ям/номером, скрол списку, клік по контакту → профіль.
- **Профіль контакту** — кнопка «Назад» → список; кнопка «Подзвонити» → екран дзвінка.
- **Екран дзвінка** — таймер тривалості, кнопка «Завершити дзвінок» → повернення до списку контактів.

## Структура проєкту

- `docs/screen-specs-phone-calls.md` — специфікація екранів (БА).
- `index.html`, `styles.css`, `app.js` — інтерактивний прототип (мобільний фрейм 375px).
